# 🚀 Space Mission Control – Console-Based Adventure

## 📌 Project Overview

**Space Mission Control** is a fun console-based JavaScript game where the player becomes a junior astronaut preparing for a mission to Mars.

The user interacts using `prompt()` and `alert()` to:

* Prepare the spaceship 🛰️
* Manage mission resources
* Complete mission tasks
* Earn mission points ⭐

This project demonstrates core JavaScript concepts in an interactive way.

---

## 🎮 Features

### 1. Mission Start

* Takes user input (name, age, favorite planet)
* Displays a personalized greeting using template literals

### 2. Spaceship Setup

* Uses **arrays** and **objects** to manage supplies
* Tracks quantity and status of each resource

### 3. Mission Tasks

* Interactive decision-making using `prompt()`
* Uses:

  * `if/else`
  * `switch` (optional logic)
  * `loops`
* Users complete **3 missions**
* Points are awarded for successful actions

### 4. Resource Refill

* Custom function: `refillSupply()`
* Allows user to refill spaceship resources
* Uses loops and input handling with `.toLowerCase()`

### 5. Mission Report

* Displays:

  * Remaining supplies
  * Total mission points
* Uses:

  * Object looping
  * Destructuring
  * String formatting

## 🛠️ How to Run the Project

1. Copy the code into a file named:

   ```
   mission.js
   ```

2. Open your browser (Chrome recommended)

3. Open Developer Console:

   * Right-click → Inspect → Console
   * OR press `Ctrl + Shift + I`

4. Paste the code into the console and press Enter

5. Follow the prompts to play 🎮
